1.	Funcionalidades de la aplicación
	Actualmente la aplicación permite realizar lo siguiente:
	•	Consultar una lista de integrantes.
	•	Visualizar la información de los Integrantes previamente seleccionados.
	•	Consultar una lista de instrumentos.
	•	Visualizar la información del instrumento previamente seleccionado.
	
2.	Pulgas o “known issues”
	El único problema conocido es el no acceso a la lista especifica de cada integrante, por el momento la funcionalidad fue deshabilitada.
3.	Pendientes del proyecto
 

Actualmente el proyecto esta en una fase inicial. La idea de este es abarcar varios módulos de trabajo que incluyan las siguientes funcionalidades:
	•	Registro y edición de tipos de instrumentos disponibles.
	•	Registro de Secciones de instrumentos.
	•	Registro de bitácora da daños para los instrumentos.
	•	Registro de eventos de la banda.
	•	Control de asistencia a eventos de la banda.
	•	Control de pagos de la Banda.
	•	Registro de productos o servicios ofrecidos por la banda.
	•	Control de seguridad y acceso por medio de un login.
	•	Utilización de Notificaciones remotas para dar a conocer los eventos
	•	Utilización de códigos QR para registrar la participación en los eventos.
	•	Utilización de mapas para localizar los eventos.

4.	Detalles de la API Utilizadas
	La aplicación utiliza los siguientes elementos:
	•	Almacenamiento de datos locales por medio de SQLite
	•	Manejo de menús desplegables para seleccionarlos módulos.
	•	Utilización de Tabs para separar el contenido de los listados y la edición de elementos.
